ESP32 Weather Arduino Displayer

This project consists of creating an application with an Arduino ESP32 and an 8x8 LED matrix. In my case, I've made a weather display that gets data from the OpenWeather API and shows the weather conditions, temperature and humidity on the display.

Functionalities

The display has 3 pages that change every 5 seconds to show these 3 variables. To simplify the process, I used the OpenWeather library, which makes it easier to make requests to the API and avoid dealing with JSON data.

Implementation

The Arduino ESP32 uses the WiFi module to connect to the Internet and get data from the OpenWeather API. The request to the API is made using the "WiFi" and "HTTPClient" libraries. The JSON_Decoder library helps to process the JSON response from the API.

Display UI

The display is too small to show all the information at once, so I created pages. There are three main pages, each with an indicator at the bottom that highlights when the corresponding page is displayed. The first page shows the weather conditions (sun, rain, storm...), the second shows the temperature and the third shows the humidity.

Indicators

At the top right, there is an LED indicator that shows whether the WiFi is connected (green) or not (red), helping the user understand why the device is not working.

Libraries useds

  • Adafruit_NeoPixel for matrix control.
  • JSON_Decoder i OpenWeather to simplify the API requests.
  • WiFi i HTTPClient to connect to internet and be able to do API requests.
